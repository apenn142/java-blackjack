# java-cardgames
## Written in Java in conjunction with JavaFX and SceneBuilder.
### jdk1.8.0_281, 
**Code is under the Group12FinalProject/src/application/ directory**

Features 4 card games including:
- Blackjack (most complete)
- Hearts
- War
- Concentration
